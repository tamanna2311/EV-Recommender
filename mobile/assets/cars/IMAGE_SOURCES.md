# EV Finder car image assets

These images are app-owned presentation assets used by the mobile frontend.

- CSV `image_url` values point to `/assets/cars/<car_id>.png`.
- The source dataset previously used many `imgd.aeplcdn.com`/CarWale CDN URLs.
- The following visible third-party plate labels were cleaned from the app assets:
  - `mg_comet_ev`
  - `mg_windsor_ev`
  - `tata_nexon_ev`
  - `hyundai_creta_electric`
  - `mahindra_xuv400`
  - `tata_curvv_ev`
  - `mahindra_be_6`
  - `mahindra_xev_9e`
  - `mini_countryman_electric`
  - `kia_ev6`
  - `bmw_i5`
- `vayve_mobility_eva` was refreshed from the current Vayve official site because the old dataset image URL no longer returned an image.
